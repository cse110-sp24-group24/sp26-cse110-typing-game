# US-05 · Language · Sprint 1

## Selecting a coding language before starting

**As a student learning JavaScript or Python**, I want to choose my language before a run, so that all the code I type is relevant to what I'm currently studying.

---

## Acceptance criteria

- [ ] Language selector appears on the pre-run screen
- [ ] At least JavaScript and Python are available for MVP
- [ ] All code snippets in the run use only the selected language's syntax
- [ ] Language choice is displayed in the HUD during the run

---

## Notes

Language selection is a top-of-funnel decision that sets the educational context for the entire run. The pre-run screen should make this choice feel intentional, not buried. Post-MVP this screen is also where difficulty and session length options would live, so build it with that expansion in mind.

## Dependencies

- Snippet library must have curated functions for each available language (see US-10)

## Definition of done

A player can select JavaScript or Python on the pre-run screen. All snippets encountered during the run match that language. The selected language label is visible in the HUD throughout the run.
