# US-02 · Difficulty Scaling · Sprint 1

## Increasing challenge as waves progress

**As a player**, I want later waves to become more difficult, so that the game remains engaging as my skills improve.

---

## Acceptance criteria

- [ ] Enemy movement speed increases over time
- [ ] Snippet complexity scales with progression
- [ ] Advanced syntax appears in later waves
- [ ] Difficulty resets on a new run
- [ ] Scaling values are configurable

---

## Notes

Difficulty progression should feel gradual instead of unfairly spiking.

## Dependencies

- Wave progression system
- Snippet complexity tagging

## Definition of done

A player experiences progressively harder gameplay as waves increase.
