# US-01 · Wave Combat · Sprint 1

## Typing code to defeat enemies

**As a player**, I want enemies to attack me using code snippets that I must type correctly, so that gameplay feels directly tied to my typing performance.

---

## Acceptance criteria

- [ ] Each enemy displays a code snippet
- [ ] Correctly typed characters damage the enemy
- [ ] Incorrect characters are highlighted immediately
- [ ] Enemy defeat animations trigger after completion
- [ ] The next enemy spawns automatically after defeat

---

## Notes

The typing loop is the foundation of the game and must feel responsive with minimal input delay.

## Dependencies

- Real-time input validation system
- Enemy spawning system

## Definition of done

A player can type through multiple enemies in sequence and defeat them through accurate input.
