# US-03 · Language Selection · Sprint 1

## Choosing programming languages before gameplay

**As a player**, I want to choose which programming language appears in the game, so that I can practice syntax relevant to my interests.

---

## Acceptance criteria

- [ ] Players can choose between supported languages
- [ ] Snippets load based on selected language
- [ ] Language choice persists during gameplay
- [ ] Unsupported languages are hidden
- [ ] Current language is visible in the HUD

---

## Notes

This feature personalizes the learning experience and supports replayability.

## Dependencies

- Snippet database
- Main menu system

## Definition of done

A player can select a language and receive matching snippets throughout gameplay.
