# US-06 · Endless Survival Mode · Sprint 2

## Playing endless wave survival

**As a player**, I want an endless survival mode, so that I can continuously challenge my typing endurance.

---

## Acceptance criteria

- [ ] Waves continue until player defeat
- [ ] Difficulty scales infinitely
- [ ] High scores are recorded
- [ ] Survival duration is tracked
- [ ] Endless mode appears in the menu

---

## Notes

This mode supports replayability and competitive score chasing.

## Dependencies

- Persistent score storage
- Difficulty scaling system

## Definition of done

A player can enter endless mode and survive for as long as possible while scores are tracked.
