# US-04 · Combo Multiplier · Sprint 2

## Rewarding consecutive accurate typing

**As a player**, I want accurate typing streaks to increase my combo multiplier, so that precision feels rewarding.

---

## Acceptance criteria

- [ ] Consecutive correct inputs increase combo count
- [ ] Combos increase score values
- [ ] Typos reset the combo
- [ ] Combo effects are visible on screen
- [ ] Audio feedback plays during high combos

---

## Notes

The combo system should encourage controlled accuracy instead of random speed typing.

## Dependencies

- Scoring system
- Accuracy tracking system

## Definition of done

A player can build and lose combos dynamically during gameplay.
