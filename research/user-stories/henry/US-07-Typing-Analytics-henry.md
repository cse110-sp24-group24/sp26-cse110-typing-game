# US-07 · Typing Analytics · Sprint 2

## Reviewing detailed typing performance

**As a player**, I want to see detailed performance analytics after each run, so that I can improve weak areas.

---

## Acceptance criteria

- [ ] WPM is displayed after a run
- [ ] Accuracy percentage is displayed
- [ ] Common mistakes are tracked
- [ ] Total enemies defeated are recorded
- [ ] Results persist between sessions

---

## Notes

Analytics should encourage improvement rather than punish mistakes.

## Dependencies

- Gameplay analytics tracking
- Save system

## Definition of done

A player can review detailed performance data after finishing gameplay.
