# US-09 · Haunted Theme · Sprint 1

## Creating a horror-themed coding atmosphere

**As a player**, I want a haunted visual aesthetic, so that the game feels memorable and immersive.

---

## Acceptance criteria

- [ ] Backgrounds use dark horror-inspired visuals
- [ ] Enemies fit the haunted theme
- [ ] UI remains readable in dark scenes
- [ ] Lighting effects react to gameplay
- [ ] Menus follow the same visual style

---

## Notes

Visual atmosphere should support gameplay readability at all times.

## Dependencies

- Art assets
- UI theme system

## Definition of done

Players experience a consistent haunted visual identity across gameplay and menus.
