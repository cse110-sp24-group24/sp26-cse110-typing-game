# US-05 - Accuracy Tracking - Sprint 1

## Understanding typing mistakes

**As a cs student**, I want to see my typing accuracy after each round so that I can understand where I need improvement.

---

## Acceptance criteria

- [ ] Accuracy percentage is shown after each run
- [ ] Mistyped words are counted correctly
- [ ] Results screen displays total mistakes
- [ ] Accuracy updates in real time during gameplay

---

## Notes

Tracking accuracy helps beginners focus on correctness instead of only speed.

## Dependencies

- Typing system must track mistakes
- Results screen must display statistics

## Definition of done

Players receive clear feedback about their typing accuracy after gameplay sessions.
