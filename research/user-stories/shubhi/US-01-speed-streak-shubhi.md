# US-02 - Combo System - Sprint 1

## Building confidence through streaks

**As a new student studying computer science**, I want the game to reward me for typing multiple words correctly in a row so that I feel more motivated to improve my consistency and speed.

---

## Acceptance criteria

- [ ] Consecutive correct words increase a visible combo streak
- [ ] Combo streak resets after a typing mistake
- [ ] Higher streaks increase score multipliers
- [ ] Player receives visual feedback when a streak milestone is reached

---

## Notes

Small rewards and streak systems help newer students stay engaged while practicing repetitive skills like typing.

## Dependencies

- Typing system must detect correct and incorrect inputs
- Score system must support multipliers
- UI system must display active streaks

## Definition of done

Players can build and lose typing streaks naturally during gameplay while seeing score bonuses tied to their consistency.
