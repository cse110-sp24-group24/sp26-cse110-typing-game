# US-05 - Pause System - Sprint 1

## Allowing players to take breaks

**As a student studying cs**, I want to pause the game during a run so that I can step away briefly without losing all my progress.

---

## Acceptance criteria

- [ ] Pressing the pause button freezes gameplay
- [ ] Timer and enemies stop while paused
- [ ] Pause menu includes resume and quit options
- [ ] Game resumes correctly from the paused state

---

## Notes

Students may need quick interruptions while playing between assignments or study sessions.

## Dependencies

- Game loop must support pausing
- UI must support a pause overlay

## Definition of done

Players can safely pause and resume a run without breaking gameplay.
