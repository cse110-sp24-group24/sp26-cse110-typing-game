# US-03 - Focus Mode - Sprint 1

## Reducing distractions for beginners

**As a student studying cs**, I want a focus mode that gets rid of visual clutter so that I can concentrate on improving my typing without feeling overwhelmed.

---

## Acceptance criteria

- [ ] Players can enable or disable focus mode in settings
- [ ] Focus mode reduces unnecessary animations and effects
- [ ] Only essential gameplay information remains visible
- [ ] Setting is saved between sessions

---

## Notes

Some new students may feel overstimulated by fast-paced visuals while learning typing fundamentals.

## Dependencies

- Settings menu must support toggles
- UI system must support simplified layouts

## Definition of done

That players can switch into a cleaner, distraction-free version of the game that improves readability and focus.
