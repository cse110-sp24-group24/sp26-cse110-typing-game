# US-04 - Coding Vocabulary Practice - Sprint 1

## Practicing real CS terminology

**As a student studying cs**, I want the game to include programming related words and phrases so that I can become comfortable/familiar with common coding terminology while practicing typing.

---

## Acceptance criteria

- [ ] Game includes a coding vocabulary word set
- [ ] Words include beginner programming terms and syntax
- [ ] Players can choose coding vocabulary mode before starting
- [ ] Word list avoids advanced terminology intended for experts

---

## Notes

Exposure to common programming words can help students become more comfortable with technical language early on.

## Dependencies

- Word database must support categorized word sets
- Menu system must allow mode selection

## Definition of done

Players can practice typing beginner-friendly programming vocabulary during gameplay.
