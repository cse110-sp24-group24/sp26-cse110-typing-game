# US-06 - Interactive beginner Tutorial - Sprint 1

## Learning how the game works

**As a cs student**, I would like a short tutorial when I first open the game so that I can quickly understand the controls and gameplay mechanics.

---

## Acceptance criteria

- [ ] First-time users automatically see the tutorial
- [ ] Tutorial explains movement, typing, and scoring
- [ ] Players can skip the tutorial
- [ ] Tutorial does not appear again after completion unless manually restarted

---

## Notes

New players should understand the game quickly without needing outside instructions.

## Dependencies

- Save system must track tutorial completion
- UI system must support guided prompts

## Definition of done

First-time players can complete or skip a beginner-friendly tutorial before starting gameplay.
