# US-08 - Difficulty Selection - Sprint 1

## Matching different skill levels

**As a CS student**, I want to choose between different difficulty levels so that I can practice at a pace that feels comfortable.

---

## Acceptance criteria

- [ ] Game includes beginner, intermediate, and advanced difficulties
- [ ] Difficulty changes enemy speed and word complexity
- [ ] Selected difficulty is displayed before gameplay starts
- [ ] Players can change difficulty between runs

---

## Notes

Not all incoming students type at the same level, so pacing flexibility is important.

## Dependencies

- Enemy system must support scalable difficulty
- Word database must support varying complexity

## Definition of done

Players can choose a difficulty level that noticeably changes gameplay pacing and challenge.
