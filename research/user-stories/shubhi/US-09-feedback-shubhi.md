# US-09 - Performance Summary - Sprint 1

## Receiving helpful feedback after gameplay

**As a CS student**, I would like a summary screen after each run so that I can understand how I performed and what I should improve.

---

## Acceptance criteria

- [ ] Summary screen appears after every run
- [ ] Screen displays score, WPM, and accuracy
- [ ] Screen highlights strongest and weakest metrics
- [ ] Player can quickly restart from the summary screen

---

## Notes

Clear feedback helps players stay aware of their progress and improvement areas.

## Dependencies

- Statistics system must track gameplay performance
- UI system must support results screens

## Definition of done

Players receive a detailed but easy-to-read summary after every gameplay session.
