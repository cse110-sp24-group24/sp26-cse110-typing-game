# US-07 · End-of-Run Statistics Screen · Sprint 2

## Review performance after a run ends

**As a player**, I want to see my performance statistics after dying, so that I can track improvement and understand how well I performed.

---

## Acceptance criteria

- [ ] End screen appears after game over
- [ ] Accuracy percentage is displayed
- [ ] Words per minute (WPM) is displayed
- [ ] Waves survived and final score are displayed

---

## Notes

Performance metrics help motivate replayability and improvement. Players should clearly understand what they did well and where they struggled.

---

## Dependencies

- Accuracy tracking system
- Score tracking system

---

## Definition of done

After a run ends, the game displays complete performance statistics to the player.
