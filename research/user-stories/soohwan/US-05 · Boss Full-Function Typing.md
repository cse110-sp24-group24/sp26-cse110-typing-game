# US-05 · Boss Full-Function Typing · Sprint 2

## Type the complete function during boss fights

**As a player**, I want the boss fight to require typing the full function from the wave, so that the boss feels like a culmination of everything I practiced earlier.

---

## Acceptance criteria

- [ ] Boss appears after all wave enemies are defeated
- [ ] Entire completed function is shown at once
- [ ] Player must type the entire function correctly
- [ ] Boss is defeated only after full completion

---

## Notes

The boss should test memory and syntax familiarity developed during the wave. It should feel like a final exam rather than a separate mechanic.

---

## Dependencies

- Function assembly system
- Boss encounter system

---

## Definition of done

The boss encounter requires successfully typing the entire function generated during the wave.
