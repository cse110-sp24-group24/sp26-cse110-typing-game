# US-03 · Wave Completion Reward · Sprint 1

## Upgrade selection after each wave

**As a player**, I want to choose from randomized upgrades after surviving a wave, so that each run feels different and rewards progression.

---

## Acceptance criteria

- [ ] Upgrade screen appears after clearing a wave
- [ ] At least 3 random upgrades are displayed
- [ ] Player can select only one upgrade
- [ ] Chosen upgrade immediately affects gameplay

---

## Notes

The upgrade system is the main roguelite element of the game. The choices should feel meaningful enough to change how the next wave plays.

---

## Dependencies

- Wave-clear detection system
- Upgrade effect system

---

## Definition of done

After every cleared wave, the player selects one upgrade and its gameplay effect becomes active immediately.
