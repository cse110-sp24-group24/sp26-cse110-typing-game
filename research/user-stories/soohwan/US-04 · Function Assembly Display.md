# US-04 · Function Assembly Display · Sprint 1

## Build a full function line by line

**As a CS student**, I want completed code lines to remain visible on screen, so that I can understand how the full function is constructed over the course of the wave.

---

## Acceptance criteria

- [ ] Defeated enemy lines remain displayed after completion
- [ ] Code lines appear in the correct order
- [ ] The full function becomes visible by the end of the wave
- [ ] Previously completed lines are visually distinct from the active line

---

## Notes

The educational value comes from seeing how smaller snippets combine into a full function. Removing completed lines would weaken the learning aspect.

---

## Dependencies

- Enemy typing system
- Persistent code display UI

---

## Definition of done

Each completed line remains visible and the player can see the entire completed function by the end of the wave.
