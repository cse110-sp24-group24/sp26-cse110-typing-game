# US-06 · Haunted Background Atmosphere · Sprint 2

## Immersive haunted environment visuals

**As a player**, I want the game environment to feel haunted and atmospheric, so that the horror theme enhances the tension of gameplay.

---

## Acceptance criteria

- [ ] Background contains haunted visual elements
- [ ] Fog, shadows, or ambient effects are visible
- [ ] Environment visuals remain readable during typing
- [ ] Theme remains visually consistent across waves

---

## Notes

The horror atmosphere should support gameplay without making the code unreadable. Style should reinforce tension while maintaining clarity.

---

## Dependencies

- Background rendering system
- Art asset pipeline

---

## Definition of done

Gameplay occurs in a visually consistent haunted environment that supports the horror tone without reducing readability.
