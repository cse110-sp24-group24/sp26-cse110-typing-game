# US-01 · Onboarding · Sprint 2

## Learn how to play the game

**As a new CS student**, I want a short first-run tutorial that explains wave typing, the function panel, the boss, and upgrade pick in plain language, so that I can learn how to learn essentially

---

## Acceptance criteria

- [ ] On first visit, a guided sequence plays before real difficulty spikes
- [ ] The tutorial uses one tiny real snippet (same language the player selected) and names each UI region once (input, target line, function assembly, health)
- [ ] The player completes a short code snippet all the way up to the boss
- [ ] “Skip tutorial” is available without losing access to help later (e.g., a “?” icon reopens key tips)

---

## Notes

The tutorial’s job is cognitive load reduction, and so that the player understands why they lost

## Dependencies

- Language selection exists and fixes the active language before the first wave or tutorial snippet loads
- Core wave typing and a function assembly area that updates as the player progresses

## Definition of done

A first-time player can finish the guided sequence, start a real run, and be able correctly predict game flow.
