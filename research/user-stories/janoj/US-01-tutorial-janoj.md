# US-01 · Onboarding · Sprint 1

## Learning the basics — guided tutorial

**As a new CS student**, I want a short guided tutorial that teaches me how to type to attack enemies, so that I can understand how the game works before starting a real run.

---

## Acceptance criteria

- [ ] The game includes a tutorial mode accessible from the main menu
- [ ] The tutorial explains how enemies approach the tower and how typing defeats them
- [ ] The tutorial introduces at least one HTML, one CSS, and one JavaScript token
- [ ] The player must successfully complete at least three practice actions before finishing the tutorial
- [ ] The tutorial uses simple language and does not assume prior experience with the game

---

## Notes

Because the target player is a new CS student, the tutorial should explain both the gameplay loop and the idea that the words being typed are real coding terms. The tutorial should feel low-pressure and should avoid overwhelming the player with too many mechanics at once.

## Dependencies

- Main menu flow
- Core typing and enemy systems
- Beginner token set for HTML, CSS, and JavaScript

## Definition of done

A first-time player can start the tutorial, complete the guided steps, and leave with a clear understanding of how to type tokens to defend the tower.
