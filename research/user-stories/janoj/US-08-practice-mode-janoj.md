# US-08 · Practice Mode · Sprint 3

## Practicing without pressure — sandbox syntax mode

**As a new CS student**, I want a no-fail practice mode, so that I can get comfortable with HTML, CSS, and JavaScript tokens before playing the survival mode.

---

## Acceptance criteria

- [ ] The game includes a practice mode separate from the main run mode
- [ ] Practice mode allows players to type tokens without tower health loss or wave failure
- [ ] The player can choose at least one language category to practice
- [ ] Practice mode displays whether typed tokens are correct
- [ ] Practice mode can be entered from the main menu

---

## Notes

Some new CS students may want to use the project more like a study tool before trying the more game-like experience. This mode supports learning and accessibility.

## Dependencies

- Main menu navigation
- Token/content category system
- Core typing validation

## Definition of done

Players can enter a safe practice environment and type language-specific tokens without the pressure of enemies damaging the tower.
