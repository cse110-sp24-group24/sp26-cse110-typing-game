# US-09 · Learning Support · Sprint 3

## Understanding what I typed — token meaning hints

**As a new CS student**, I want short explanations or hints for syntax tokens, so that I can connect the words I type to what they mean in HTML, CSS, or JavaScript.

---

## Acceptance criteria

- [ ] The game can show a short hint or explanation for selected tokens
- [ ] Hints use beginner-friendly language
- [ ] Hints are available in tutorial, practice mode, or another non-disruptive context
- [ ] Hints do not block the player’s ability to continue playing
- [ ] At least some HTML, CSS, and JavaScript tokens include hints

---

## Notes

This helps the project feel educational rather than purely mechanical. Hints should be lightweight, like “div: a generic HTML container” or “margin: controls outside spacing.”

## Dependencies

- Token metadata structure
- UI component for hint display
- Tutorial or practice-mode integration

## Definition of done

Players can access short, understandable explanations for at least part of the syntax they are typing.
