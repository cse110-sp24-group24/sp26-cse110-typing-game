# US-07 · Narrative · Sprint 3

## Unlocking haunted story moments through progress

**As a player**, I want each completed wave to reveal a small piece of the haunted story, so that I feel motivated to continue beyond improving my score.

---

## Acceptance criteria

- [ ] Completing a wave unlocks a short story beat, visual clue, or room progression
- [ ] Story moments are brief and do not block the player from continuing for long
- [ ] The haunted narrative connects to the typing theme or code recovery theme
- [ ] Previously unlocked story beats can be reviewed from a menu or run summary

---

## Notes

The game can use narrative as a second motivation loop. Instead of only asking players to chase numbers, each wave can feel like clearing a room in a haunted mansion or recovering corrupted code from a cursed terminal. The story should be light enough to support gameplay without becoming a separate visual novel.

## Dependencies

- Wave completion events must trigger unlocks
- UI needs a small story panel or transition space
- Theme direction should define the haunted setting and tone

## Definition of done

After completing waves, players unlock short haunted story moments that add progression and can be reviewed later without interrupting core typing gameplay.
