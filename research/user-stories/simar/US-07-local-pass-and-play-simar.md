# US-07 - Collaboration - Sprint 2

## Practicing with a friend in alternating turns

**As a new CS student**, I want a local pass-and-play mode where two players alternate waves, so that I can practice with classmates without online multiplayer complexity.

---

## Acceptance criteria

- [ ] Two-player local mode is selectable from main menu
- [ ] Players alternate full waves on a shared keyboard
- [ ] Scoreboard tracks each player's wave clears and accuracy
- [ ] Session ends after a configurable number of rounds

---

## Notes

New students often learn better with peers. A simple local mode supports accountability and friendly competition while staying technically lightweight.

## Dependencies

- Session manager must support alternating turn ownership
- Scoreboard UI must render per-player metrics

## Definition of done

Two players can run an alternating-wave session, see separate stats, and finish after a selected round count.
