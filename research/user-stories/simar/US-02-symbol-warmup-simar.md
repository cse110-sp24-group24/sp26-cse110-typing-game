# US-02 - Learning - Sprint 1

## Understanding symbols before pressure starts

**As a new CS student**, I want a short symbol warmup for keys like `{}`, `()`, `[]`, `;`, and `:` before each run, so that I can build confidence with programming punctuation.

---

## Acceptance criteria

- [ ] A 15-20 second symbol warmup is available from pre-wave UI
- [ ] Warmup includes mixed symbol patterns used in the selected language
- [ ] Warmup does not affect score, lives, or run progress
- [ ] Player can disable warmups in settings

---

## Notes

Many beginners struggle more with symbols than letters. A lightweight warmup helps reduce early errors and improves readiness for real code lines.

## Dependencies

- Language selection must provide language-specific symbol sets
- Settings menu must store warmup toggle preference

## Definition of done

Players can run or skip a short symbol-only warmup before gameplay and enter wave 1 without affecting game state.
