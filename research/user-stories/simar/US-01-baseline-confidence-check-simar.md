# US-01 - Onboarding - Sprint 1

## Starting with a confidence check

**As a new CS student**, I want a quick baseline typing check before my first run, so that the game can start me at a difficulty that feels fair instead of overwhelming.

---

## Acceptance criteria

- [ ] First-time players see an optional 30-second baseline check before wave 1
- [ ] Baseline check records words per minute and accuracy
- [ ] Player can skip and use default beginner difficulty
- [ ] Recommended starting preset is shown after the check

---

## Notes

Incoming students have very different typing backgrounds. A fast baseline reduces frustration and helps the game personalize early pacing without asking players to guess settings.

## Dependencies

- Input system must capture timing and mistakes during the check
- Difficulty system must support selecting a beginner-friendly preset

## Definition of done

A first-time player can complete or skip a short baseline check and immediately start with a clearly recommended difficulty preset.
